섯다 카드게임 서버 & 클라이언트

제작자 pureless29입니다.
서버는 Windows 10 OS에서 Visual Studio 2019 Community 버전을 이용해 개발하였습니다.
클라이언트는 동일한 환경의 Android Studio 3.5 버전을 이용해 개발하였습니다.

해당 서버 프로그램은 Windows XP 이상의 환경에서 동작합니다. (IOCP 지원)
서버 프로그램을 실행하면 콘솔창에 해당 서버의 공인 IP 주소와,
해당 서버 소켓으로 열린 포트번호가 출력됩니다.

클라이언트는 저희 집 컴퓨터 IP 주소로 설정된 서버로 접속하게 만들어 놓았습니다.
직접 실행해보고자 하시는 분은 IDE에서 NetworkManager.java의 47번째 줄에 있는
hostname(IP주소)를 수정하시면 됩니다.

========================
프로그램의 대략적인 개요가 궁금하신 분들은 명세서 ppt와 실행 스크린샷을 보시면 되겠습니다.
========================

이 프로그램은 개인 공부 및 포트폴리오 용으로 개발된 프로그램으로,
어떠한 상업적, 영리적 목적으로 이용되지 않을 것임을 밝힙니다.
